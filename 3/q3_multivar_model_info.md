# 扩展混合效应模型结果
==================================================

## 模型形式
Y_ij = (β₀ + u_i) + β₁·Week_ij + β₂·BMI_ij + β₃·Age_ij + β₄·GC_ij + ... + ε_ij

其中：
- Y_ij: 第i位孕妇第j次检测的Y染色体浓度
- β₀: 固定截距（全体平均基线）
- u_i: 第i位孕妇的随机截距（个体差异）
- β₁, β₂, β₃, β₄, ...: 各变量的固定效应系数（包括GC含量）
- ε_ij: 残差项
- 注意：孕妇代码仅用于分组，不作为自变量

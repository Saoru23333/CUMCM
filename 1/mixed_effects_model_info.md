# 混合效应模型结果
==================================================

## 模型形式
Y_ij = (β₀ + u_i) + β₁·Week_ij + β₂·BMI_ij + ε_ij

其中：
- Y_ij: 第i位孕妇第j次检测的Y染色体浓度
- β₀: 固定截距（全体平均基线）
- u_i: 第i位孕妇的随机截距（个体差异）
- β₁: 孕周的固定效应系数
- β₂: BMI的固定效应系数
- ε_ij: 残差项
